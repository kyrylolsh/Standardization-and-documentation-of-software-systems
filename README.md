### 💻 Базові команди
Всі команди для розробки та тестування виконуються в папці `kop_whack_a_mole`:

1.  **Встановлення залежностей:** 
    npm install
   
2.  **Запуск гри (локальний сервер):** 
    npm start
    
3.  **Запуск середовища Storybook:** 
    npm run storybook
    
4.  **Генерація технічної документації:** 
    npx jsdoc src/ -d ../docs
